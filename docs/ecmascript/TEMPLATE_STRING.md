# Template string

Template strings = template literals

Using backstics = `

Example above

```js
const wind = "10 m/s";
const content = `${wind} wind`;

console.log(content);
```

Console output: 10 m/s wind

Another example

```js
const number = 5;
const content = `Number ${number}`;

console.log(content);
```

Console output: Number 5
